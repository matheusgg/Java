Joshua J. Bloch (born August 28, 1961) is a software engineer, formerly employed at Google, and a 
technology author. He led the design and implementation of numerous Java platform features, including 
the Java Collections Framework, the java.math package, and the assert mechanism.[1] He is the author 
of the programming guide Effective Java, which won the 2001 Jolt Award,[2] and is a co-author of two 
other Java books, Java Puzzlers (2005) and Java Concurrency In Practice (2006).
